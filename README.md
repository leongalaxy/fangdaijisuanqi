# 房贷计算器 2026年最新版

一个功能齐全的房贷计算器网页应用，支持商业贷款、公积金贷款和组合贷款计算。

## 功能特点

- 支持三种贷款类型：
  - 商业贷款
  - 公积金贷款
  - 组合贷款

- 支持两种还款方式：
  - 等额本息还款
  - 等额本金还款

- 利率计算方式：
  - LPR利率 + BP基点
  - 固定利率

- 详细功能：
  - 实时计算月供、总还款额、总利息
  - 每月还款明细（默认显示12期，可展开查看36期）
  - 响应式设计，支持移动端访问
  - 官方查询网站链接

## 本地运行

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件
3. 或使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需要安装 http-server)
   npx http-server
   ```
4. 在浏览器中访问 `http://localhost:8000`

## 使用说明

1. 选择贷款类型（商业贷/公积金贷/组合贷）
2. 输入贷款金额和期限
3. 选择利率类型并设置利率
4. 点击"开始计算"按钮
5. 查看计算结果和每月还款明细
6. 点击"展开查看更多"查看更多期数的还款明细
